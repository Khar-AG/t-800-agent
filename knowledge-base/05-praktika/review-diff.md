## Простыми словами

**Diff** — экран «было / стало». Зелёное — добавлено, красное — удалено.

## Когда вам это нужно

После каждой работы Agent — до того как сказать «готово».

## Пошагово

1. Agent вносит правки — откроется панель изменений
2. Пролистайте каждый файл
3. **Accept** — принять файл или все
4. **Reject** — отклонить
5. Непонятно? Спросите в Ask: «объясни эту правку в @file»

## Аналогия

Редактор прислал правки в договор — вы сравниваете версии track changes.

## Частые ошибки

- Accept All не глядя — риск лишнего кода
- Не заметили удаление важной строки
