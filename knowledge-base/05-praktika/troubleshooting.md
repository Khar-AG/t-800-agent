## Простыми словами

Что делать, если Agent «застрял», делает не то или кончился лимит.

## Agent ходит по кругу

1. Нажмите **Stop**
2. **Новый чат**
3. Сузьте задачу: одна цель, один файл
4. Попробуйте **Plan** вместо Agent

## Scope creep (делает слишком много)

- Явно: «Только файл X, не трогай остальное»
- Используйте `@file`

## Контекст переполнился

- Новый чат
- Краткое резюме в первом сообщении
- Subagent Explore для поиска
