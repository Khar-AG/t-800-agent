## Простыми словами

Cursor работает с **папкой** — как рабочий стол для одного проекта. Все файлы внутри — «ваш проект».

## Когда вам это нужно

Вы только что установили Cursor и не знаете, что открывать.

## Пошагово

1. Создайте папку на диске, например `Мои-проекты/мой-сайт`
2. В Cursor: **File → Open Folder** → выберите эту папку
3. Слева появится список файлов (пока пустой — это нормально)
4. Нажмите `Ctrl+I` и напишите: «Объясни эту папку. С чего начать?»

## Аналогия

Папка — как папка с документами на работе. Agent — коллега, который может заглянуть во все файлы сразу.

## Проверка

Слева в Explorer видна ваша папка. Внизу статус-бара — путь к ней.

## Частые ошибки

- Открыли один файл вместо папки — Agent не видит соседние файлы
- Папка на сетевом диске без прав — могут быть ошибки сохранения
