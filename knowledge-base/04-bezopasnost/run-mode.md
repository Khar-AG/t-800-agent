## Простыми словами

**Run Mode** — настройка, может ли Agent запускать команды в терминале и с какими ограничениями.

## Режимы (упрощённо)

| Режим | Для новичка |
|-------|-------------|
| **Auto-review** | Рекомендуется: спрашивает перед рискованным |
| **Allowlist** | Только команды из списка |
| **Run Everything** | Всё без спроса — **не для новичков** |

## Пошагово

1. Settings → Agents (или Terminal / Run Mode)
2. Выберите **Auto-review**
3. При появлении запроса на команду — читайте и одобряйте осознанно

## Частые ошибки

- Run Everything на рабочем проекте — риск удаления данных
- Одобряете команды, не понимая текста — спросите Agent «что эта команда делает?»
