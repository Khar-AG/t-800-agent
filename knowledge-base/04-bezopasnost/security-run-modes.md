# Security и Run Modes

## Простыми словами

Run Mode отвечает за то, как Cursor разрешает агенту запускать команды и инструменты. Это «ремень безопасности» для Agent.

## Основные режимы

| Режим | Для новичка | Что значит |
|-------|-------------|------------|
| Auto-review | Рекомендуется | Cursor сам оценивает риск и спрашивает при опасном |
| Allowlist | Когда понимаете команды | Разрешены только команды из списка |
| Run Everything | Не для новичков | Всё запускается без лишних вопросов |

## Пошагово

1. Откройте Settings → Agents / Terminal
2. Выберите **Auto-review**
3. Если Agent просит команду — прочитайте её
4. Если непонятно — попросите объяснить
5. Опасные команды отклоняйте

## Красные флаги

- Удаление файлов или папок
- Команды с секретами
- Деплой на production
- Изменение git history
- Массовые операции без backup
