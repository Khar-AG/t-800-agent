# Search tool — поиск по проекту

## Простыми словами

Search tool помогает Agent быстро найти нужные файлы, функции и места в коде без ручного просмотра всех папок.

## Когда вам это нужно

- «Где находится главная страница?»
- «Найди форму регистрации»
- «Где используется этот текст?»
- «Покажи, какие файлы отвечают за оплату»

## Два вида поиска

| Вид | Простыми словами | Когда полезен |
|-----|------------------|---------------|
| Semantic search | По смыслу | «Где логика входа?» |
| Точный поиск | По словам | «Найди `submitForm`» |

## Хороший промпт

```text
Найди, где в проекте реализована форма обратной связи. Сначала только объясни файлы, ничего не меняй.
```

## Частые ошибки

- Просить сразу править, не поняв структуру
- Не уточнить, что нужно только найти
- Давать слишком широкий запрос «проверь весь проект»
