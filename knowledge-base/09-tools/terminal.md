# Terminal tool — команды в Cursor

## Простыми словами

Terminal tool — это когда Agent может запускать команды в терминале: установить зависимости, запустить тесты, собрать проект.

## Когда вам это нужно

- Проверить, работает ли проект
- Запустить `npm install`, `npm test`, `npm run build`
- Посмотреть ошибки после запуска приложения

## Аналогия

Терминал — как пульт управления мастерской. Agent может нажимать кнопки, но опасные кнопки лучше подтверждать вручную.

## Пошагово

1. Попросите Agent: «Запусти проверки проекта»
2. Agent предложит команду
3. Прочитайте, что команда делает
4. Если команда понятна — разрешите
5. Если непонятна — спросите: «Объясни команду простыми словами»
6. После выполнения попросите Agent кратко объяснить результат

## Безопасность

Новичкам лучше держать **Auto-review**. Не включайте **Run Everything**, пока не понимаете последствия.

## Частые ошибки

- Одобрять команды удаления, не читая
- Запускать install/build не из той папки
- Не смотреть вывод ошибок
