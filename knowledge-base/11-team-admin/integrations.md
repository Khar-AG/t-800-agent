# Integrations — внешние сервисы

## Простыми словами

Integrations подключают Cursor к сервисам команды: GitHub, Slack, GitLab, Linear, Jira, Notion и другим.

## Когда это нужно

- Agent должен работать с GitHub issue или PR
- Нужны уведомления в Slack
- Команда ведёт задачи в Linear/Jira
- Нужно связать Cursor с рабочими процессами компании

## Чем отличается от MCP

| MCP | Integrations |
|-----|--------------|
| Инструменты и данные для агента | Подключение сервисов аккаунта/команды |
| Часто настраивается в проекте | Часто настраивается в Dashboard |
| Может быть кастомным | Обычно официальный коннектор |

## Для новичка

Не подключайте всё сразу. Подключайте только сервис, который нужен для конкретной задачи.
