# Teams и Dashboard

## Простыми словами

Dashboard — кабинет команды Cursor: участники, расходы, shared canvases, integrations, API keys и настройки организации.

## Когда вам это нужно

- Проверить, есть ли вы в team
- Настроить Shared Canvases
- Посмотреть usage/spending
- Управлять участниками
- Включить интеграции

## Что объяснять новичку

Если человек один, ему часто хватает личного Cursor. Team нужен, когда есть коллеги, общий доступ, billing или shared features.

## Частые места

| Раздел | Зачем |
|--------|-------|
| Members | Кто в команде |
| Usage / Spending | Расходы и лимиты |
| Shared Canvases | Опубликованные canvas |
| Integrations | GitHub, Slack и др. |
| API Keys | Ключи для API |
