# Bugbot и Security Agents

## Простыми словами

Bugbot и Security Agents — помощники для командной проверки кода: найти баги, риски безопасности и проблемы в PR.

## Когда это нужно

- Есть pull request
- Нужно независимое ревью
- Проект связан с auth, платежами, персональными данными
- Команда хочет автоматическую проверку перед merge

## Новичку важно

Это не замена вашему diff review. Сначала научитесь проверять изменения Agent, потом подключайте Bugbot на PR.

## Частые ошибки

- Считать Bugbot гарантией отсутствия багов
- Игнорировать human review
- Запускать security review без описания контекста
