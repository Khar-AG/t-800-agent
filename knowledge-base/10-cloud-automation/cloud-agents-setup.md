# Cloud Agents — агенты в облаке

## Простыми словами

Cloud Agent — это Agent, который работает не в вашем открытом Cursor, а в облаке Cursor. Он может выполнять задачу в репозитории, пока ваш компьютер выключен.

## Когда это нужно

- Длинная задача на отдельной ветке
- Работа через GitHub
- Фоновая автоматизация
- Командная разработка

## Что важно новичку

Сначала освойте обычный Desktop Agent. Cloud Agents требуют понимания Git, веток, секретов и окружений.

## Пошагово

1. Подключите репозиторий
2. Откройте Cloud Agents на сайте Cursor
3. Дайте задачу с чётким результатом
4. Проверьте diff/PR
5. Не кладите секреты в репозиторий
