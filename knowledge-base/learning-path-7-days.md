# Cursor за 7 дней

Мини-курс для человека без разработки. Каждый день — один результат, не теория ради теории.

## День 1 — открыть Cursor и не бояться

**Результат:** открыта папка, Agent объяснил проект.

Промпт:

```text
Я новичок. Объясни эту папку простым языком: что здесь главное и какие 3 файла открыть первыми. Пока ничего не меняй.
```

## День 2 — Agent и diff

**Результат:** одна маленькая правка и понимание зелёного/красного diff.

Промпт:

```text
Предложи одну безопасную правку для тренировки. Сначала объясни, что изменишь, и жди моего подтверждения.
```

## День 3 — Ask, Plan, Debug

**Результат:** пользователь понимает, какой режим выбрать.

- Ask — спросить и посмотреть
- Plan — сначала согласовать план
- Agent — сделать
- Debug — найти причину ошибки

## День 4 — Rules

**Результат:** первое правило поведения для Agent.

Пример:

```text
Всегда отвечай мне на русском и объясняй сложные действия простыми словами.
```

## День 5 — Skills

**Результат:** понимание, что skill — это рецепт повторяемой работы.

Пример: «каждый раз проверяй SEO-текст по одному чеклисту».

## День 6 — MCP

**Результат:** понимание MCP как «переходника» к внешним сервисам.

Не подключайте сразу всё. Начните с одного сервиса и одного тестового вызова.

## День 7 — Canvas и автоматизация

**Результат:** первый визуальный отчёт или простой automation-план.

Промпт:

```text
Сделай Canvas-отчёт по тому, чему я научился в Cursor за неделю: что умею, что ещё настроить, какие риски.
```

## Финальная проверка

- Я понимаю, когда использовать Ask/Plan/Agent/Debug
- Я видел diff и могу откатить checkpoint
- Я знаю разницу между rule, skill, subagent и MCP
- Я не даю агенту опасные действия без проверки
