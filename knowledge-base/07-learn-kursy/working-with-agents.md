## Простыми словами

Официальный курс: как формулировать задачи, давать контекст и не раздувать scope.

## Ключевые идеи

1. **Обвязка:** Instructions + Tools + Model
2. **Промпт с @file** — указывайте файлы явно
3. **Scope creep** — режьте задачу на части
4. **Human-in-the-loop** — вы проверяете результат

## Эффективный промпт (шаблон)

```
@путь/к/файлу
Цель: [одно предложение]
Ограничения: не трогать [X]
Проверка: [как понять что готово]
```
