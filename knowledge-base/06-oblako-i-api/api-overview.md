## Простыми словами

**API** — программный доступ к данным Cursor для скриптов и дашбордов. Большинство API — для **команд Enterprise**.

## Не пугайтесь в день 1

Новичку достаточно Agent в интерфейсе. API — когда автоматизируете на уровне компании.

## Что есть (кратко)

| API | Кому |
|-----|------|
| Admin / Analytics | Enterprise команды |
| Cloud Agents API | Бета, автоматизация агентов |
| TypeScript / Python SDK | Разработчики интеграций |

## Аутентификация

API-ключ в Dashboard → API Keys. Формат `crsr_...`. Не кладите ключ в Git.

## Когда вернуться сюда

- Уже уверенно работаете в Agent
- Нужен дашборд использования команды
- Нужны агенты по расписанию через API
