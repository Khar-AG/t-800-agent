## Простыми словами

**Cloud Agents** — те же помощники, но в браузере на [cursor.com/agents](https://cursor.com/agents), без открытого Cursor на компьютере.

## Когда вам это нужно

Работаете с телефона/планшета или хотите фоновую задачу в облаке.

## Не для первого дня

Сначала освойте Desktop: папка, `Ctrl+I`, diff, checkpoints.
