## Простыми словами

**@** в чате Agent — способ сказать «смотри сюда»: файл, папка, правило, документация.

## Когда вам это нужно

Agent не угадывает, какой файл важен — укажите явно.

## Примеры

- `@index.html` — этот файл
- `@src/components/` — папка
- `@my-rule` — правило из rules
- `@Docs` — документация (если подключена)

## Пошагово

1. В поле Agent начните печатать `@`
2. Выберите из списка файл или папку
3. Допишите задачу: «Исправь опечатку в заголовке»

## Эффективный промпт

```
@README.md Объясни этот файл простым языком для маркетолога.
```

## Частые ошибки

- Не указали файл — Agent ищет сам (дольше и может ошибиться)
- Слишком много @ сразу — переполнение контекста
