# Debug Mode — разбор сложных ошибок

## Простыми словами

Debug Mode — режим для ситуаций «что-то сломалось, непонятно где». Agent собирает факты: ошибки, логи, поведение приложения.

## Когда выбирать Debug

- Есть ошибка, но причина непонятна
- Тесты падают
- Приложение запускается, но работает странно
- Нужна проверка в браузере или терминале

## Пошагово

1. Переключитесь в Debug
2. Дайте точный симптом: что нажали, что ожидали, что получили
3. Вставьте ошибку или лог
4. Разрешайте только понятные диагностические команды
5. Попросите: «Сначала найди причину, потом предложи минимальный фикс»

## Хороший промпт

```text
Кнопка отправки формы ничего не делает. Найди причину через Debug Mode, но сначала не меняй файлы.
```

## Частые ошибки

- Просить «исправь всё» без описания симптома
- Не прикладывать текст ошибки
- Одобрять опасные команды вместо диагностики
