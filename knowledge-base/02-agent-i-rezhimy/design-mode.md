# Design Mode — правки интерфейса через выделение

## Простыми словами

Design Mode помогает править интерфейс визуально: вы показываете Cursor, что именно на экране нужно изменить, а не описываете всё словами.

## Когда это нужно

- Нужно поправить UI
- Трудно объяснить элемент словами
- Нужно указать конкретный блок на странице или canvas

## Аналогия

Как дать дизайнеру скриншот и обвести красным: «вот этот блок сделать шире».

## Пошагово

1. Откройте страницу или canvas
2. Включите Design Mode, если он доступен
3. Выделите нужный элемент
4. Напишите короткую правку
5. Проверьте diff и визуальный результат

## Частые ошибки

- Просить «сделай красиво» без конкретного элемента
- Не проверять адаптив после визуальной правки
- Путать Design Mode с полноценным Figma
