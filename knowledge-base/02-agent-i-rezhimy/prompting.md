# Prompting — как ставить задачи Cursor

## Простыми словами

Промпт — это техническое задание. Чем яснее вы описали результат и ограничения, тем меньше Agent будет гадать.

## Формула хорошего промпта

```text
Цель: что должно получиться.
Контекст: какие файлы смотреть через @.
Ограничения: что не трогать.
Проверка: как понять, что готово.
```

## Пример

```text
@src/pages/home.tsx
Цель: заменить текст в hero-блоке на более понятный для новичков.
Ограничения: не менять стили и компоненты.
Проверка: покажи diff и объясни, что изменил.
```

## Частые ошибки

- «Сделай красиво» без критериев
- Не указали файлы через `@`
- Не сказали, что нельзя трогать
- Смешали текст, дизайн и деплой в один запрос
