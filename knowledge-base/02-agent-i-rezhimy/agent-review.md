# Agent Review — как проверять изменения агента

## Простыми словами

Agent Review — это привычка не принимать работу ИИ вслепую: посмотреть diff, запустить проверки и понять, что изменилось.

## Мини-чеклист

- Что изменилось в файлах?
- Есть ли лишние правки?
- Не удалены ли важные строки?
- Прошли ли тесты или сборка?
- Agent объяснил, почему так сделал?

## Пошагово

1. После работы Agent откройте diff
2. Пролистайте каждый изменённый файл
3. Попросите: «Объясни эти изменения простыми словами»
4. Запустите проверки проекта
5. Если не нравится — отклоните файл или восстановите checkpoint

## Хороший промпт

```text
Перед тем как я приму diff, объясни по файлам: что изменено, зачем, и какие риски проверить.
```
