# Agents Window — окно агентов

## Простыми словами

Agents Window — место, где вы открываете и ведёте чаты с Agent, переключаете задачи и работаете с несколькими агентными сессиями.

## Когда вам это нужно

- Создать новый чат Agent
- Вернуться к прошлой задаче
- Открыть canvas или вкладку агента
- Разделить разные задачи по разным сессиям

## Правило для новичка

Одна большая задача — один отдельный чат. Не смешивайте «объясни проект», «сделай форму» и «запусти деплой» в одной длинной беседе.

## Пошагово

1. Откройте Agents Window
2. Создайте новый агентный чат для новой задачи
3. Назовите задачу понятным первым сообщением
4. Если тема изменилась — лучше новый чат
5. Старые чаты используйте как историю, но не как бесконечную память

## Частые ошибки

- Продолжать новую тему в старом чате
- Терять контекст, потому что не указали файлы через `@`
- Ожидать, что Agent помнит все прошлые проекты
