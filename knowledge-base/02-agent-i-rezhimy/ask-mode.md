# Ask Mode — спросить без правок

## Простыми словами

Ask Mode — режим «объясни и покажи», где Cursor читает проект, но не меняет файлы.

## Когда выбирать Ask

- Вы впервые открыли чужой проект
- Нужно понять архитектуру
- Хотите спросить «где что лежит»
- Боитесь случайных изменений

## Пошагово

1. Переключитесь в Ask
2. Укажите файл или папку через `@`
3. Спросите простым языком
4. Попросите карту файлов или краткое объяснение
5. Если готовы к правкам — переходите в Plan или Agent

## Пример

```text
@src/ Объясни структуру проекта для новичка. Ничего не меняй.
```
