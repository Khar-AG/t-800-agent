# Глоссарий

| Термин | Простыми словами |
|--------|------------------|
| **Agent** | ИИ-помощник, который сам открывает файлы, пишет код и запускает команды |
| **Ask** | Режим «только спросить» — смотрит, но не меняет файлы |
| **Plan** | Сначала план, вы одобряете — потом правки |
| **Debug** | Режим поиска сложных ошибок с доказательствами |
| **Tab** | Серое автодополнение кода при наборе (как умная подсказка) |
| **Diff** | Сравнение «было / стало» — что агент изменил |
| **Checkpoint** | Снимок файлов до правки; можно откатить |
| **Canvas** | Интерактивный отчёт/дашборд рядом с чатом (не простыня текста) |
| **Shared Canvas** | Опубликованный canvas по ссылке для команды (только просмотр) |
| **Publish** | Кнопка публикации Shared Canvas в панели canvas |
| **Rule** | Постоянная инструкция для агента (стиль, язык, запреты) |
| **Skill** | Рецепт для повторяемой задачи (чеклист, workflow) |
| **Subagent** | Узкий помощник в отдельном «окне» (исследование, shell) |
| **MCP** | Подключение внешних инструментов (браузер, API, docs) |
| **Run Mode** | Как агенту разрешать команды в терминале |
| **Auto-review** | Безопасный режим: агент спрашивает перед опасными командами |
| **Allowlist** | Список разрешённых команд |
| **Terminal tool** | Инструмент Agent для запуска команд проекта |
| **Browser tool** | Инструмент Agent для открытия страниц, кликов и скриншотов |
| **Search tool** | Поиск по проекту: по смыслу или точному тексту |
| **Cloud Agent** | Агент в браузере на cursor.com/agents |
| **Automation** | Запуск агента по расписанию или событию |
| **Hook** | Автоматическое действие Cursor при событии сессии/проекта |
| **@file** | Ссылка на файл в промпте — «смотри сюда» |
| **Контекст** | Всё, что агент «помнит» в текущем чате |
| **Scope creep** | Агент раздувает задачу — делает больше, чем просили |
| **Human-in-the-loop** | Человек проверяет результат ИИ — это норма |
| **Frontmatter** | Служебный блок `---` в начале .md файла |
| **globs** | Шаблон путей файлов (например `**/*.md`) |
| **alwaysApply** | Правило действует в каждом чате |
| **Task** | Вызов субагента из главного агента |
| **Explore** | Встроенный субагент для поиска по проекту |
| **Bash** | Встроенный субагент для команд терминала |
| **Browser** | Встроенный субагент для работы с веб-страницей |
| **IDE** | Среда разработки — программа, где вы работаете с кодом |
| **Репозиторий / папка проекта** | Папка с файлами, которую вы открыли в Cursor |
| **Промпт** | Ваше сообщение агенту — задание |
| **Модель** | Какой ИИ «мозг» используется (GPT, Claude и т.д.) |
| **Лимит контекста** | Сколько текста помещается в одном чате |
| **Enterprise API** | API для команд — не нужен в первый день |
| **SDK** | Библиотека для запуска Cursor из собственного кода |
| **CLI** | Управление Cursor из командной строки |
| **Bugbot** | Автоматическое ревью багов в pull request |
| **Security Agent** | Проверка кода на риски безопасности |
| **Integration** | Подключение GitHub, Slack, Linear и других сервисов |
