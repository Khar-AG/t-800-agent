# Cloud Agents API

## Простыми словами

Cloud Agents API позволяет создавать и управлять облачными агентами программно: через HTTP-запросы и API-ключи.

## Когда это нужно

- У компании есть свой портал задач
- Нужно запускать агентов по событиям
- Нужна интеграция с внутренними системами

## Не для первого дня

Если вы только начали с Cursor, API не нужен. Сначала: Agent в IDE → Rules/Skills → MCP → Cloud Agents → только потом API.

## Безопасность

- API-ключи не хранить в Git
- Минимальные права доступа
- Лимиты и обработка 429
- Логи ошибок
