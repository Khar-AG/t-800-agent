# Cursor SDK — TypeScript и Python

## Простыми словами

SDK — набор библиотек, чтобы запускать агентов Cursor из собственного кода: TypeScript или Python.

## Когда это нужно

- Вы строите свой инструмент поверх Cursor
- Нужно программно запускать агента
- Есть backend/скрипт/пайплайн

## Почему это продвинутый уровень

Нужны навыки программирования, API-ключи и понимание окружений. Обычному новичку SDK не нужен.
