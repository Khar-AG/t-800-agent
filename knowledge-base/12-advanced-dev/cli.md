# Cursor CLI и headless-режим

## Простыми словами

CLI — способ запускать Cursor/Agent из командной строки, без обычного интерфейса IDE.

## Когда это нужно

- Автоматизация в CI/CD
- Скрипты разработчиков
- Headless-задачи без окна Cursor
- Интеграции с GitHub Actions

## Почему не для первого дня

CLI требует понимания терминала, git, окружения и прав доступа. Новичку лучше начинать с Agent в IDE.
