# Wizard — Canvas и Shared Canvas

## Цель

Помочь новичку создать отчёт/дашборд в Canvas и понять, можно ли поделиться им с командой.

## Шаги

1. Сформулируйте, какой отчёт нужен
2. Попросите Agent: «Сделай это в Canvas»
3. Проверьте данные и подписи
4. Если нужна ссылка команде — ищите Publish в canvas toolbar
5. Если Publish нет — проверьте team, тариф, Privacy Mode и тип canvas
6. При необходимости отправьте screenshot или файл `.canvas.tsx`
7. Не делитесь отчётом до проверки цифр

## Готовый промпт

```text
Сделай Canvas-отчёт по теме [тема]. Он должен быть понятен человеку без разработки: заголовки, таблица, выводы и что делать дальше.
```

## Проверка

Canvas открыт рядом с чатом, данные понятны, ссылка Publish доступна только если выполнены условия Shared Canvases.
