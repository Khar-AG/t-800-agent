# Wizard — подключить MCP

## Цель

Помочь новичку подключить внешний инструмент без утечки секретов.

## Шаги

1. Назовите сервис: браузер, docs, CMS, CRM, Wordstat и т.п.
2. Найдите официальную инструкцию MCP
3. Проверьте, нужны ли API keys
4. Секреты храните в env/secrets, не в Git
5. Добавьте MCP конфигурацию
6. Перезапустите Cursor
7. Проверьте в Agent: «Какие MCP tools доступны?»

## Готовый промпт

```text
Помоги подключить MCP для [сервис]. Объясняй простым языком. Сначала проверь, какие секреты нужны и куда их безопасно положить.
```

## Проверка

Agent видит tools, тестовый вызов проходит, секреты не лежат в репозитории.
