# Wizard — первая автоматизация

## Цель

Превратить повторяемую задачу в rule/skill/automation без риска.

## Шаги

1. Опишите повторяемую задачу одним предложением
2. Определите: это стандарт поведения или пошаговый рецепт?
3. Если стандарт — создайте rule
4. Если рецепт — создайте skill
5. Протестируйте вручную в Agent
6. Только потом думайте об Automations или Hooks
7. Проверьте безопасность Run Mode

## Готовый промпт

```text
Я хочу автоматизировать повторяемую задачу: [описание].
Сначала помоги решить, нужен rule, skill, MCP или automation. Не создавай файлы, пока я не подтвержу.
```

## Проверка

Есть один понятный rule/skill, а не огромная автоматизация сразу.
