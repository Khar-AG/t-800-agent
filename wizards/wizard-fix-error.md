# Wizard — исправить ошибку

## Цель

Научить новичка не паниковать и использовать Debug Mode.

## Шаги

1. Скопируйте точный текст ошибки
2. Опишите, что делали перед ошибкой
3. Переключитесь в Debug Mode
4. Попросите сначала найти причину, не менять файлы
5. Разрешайте только понятные диагностические команды
6. После причины попросите минимальный фикс
7. Проверьте тест/сценарий повторно

## Готовый промпт

```text
У меня ошибка: [текст ошибки].
Я сделал: [шаги].
Найди причину через Debug Mode. Сначала не меняй файлы, только объясни гипотезу и план проверки.
```

## Проверка

Есть причина ошибки, минимальный фикс и способ проверить, что проблема ушла.
