# Wizard — первый проект

## Цель

Помочь новичку открыть папку, понять структуру и сделать одну безопасную правку.

## Шаги

1. Откройте папку проекта в Cursor
2. Ask: «Объясни структуру этой папки простым языком»
3. Попросите карту главных файлов
4. Выберите одну маленькую правку
5. Agent: «Сделай только эту правку, не трогай остальное»
6. Проверьте diff
7. Если не нравится — Restore Checkpoint

## Готовый промпт

```text
Я новичок. Объясни эту папку простым языком: что здесь главное, какие файлы открыть первыми и какую одну безопасную правку можно сделать для тренировки. Пока ничего не меняй.
```

## Проверка

Пользователь знает, где главный файл, видел diff и понимает, как откатить.
