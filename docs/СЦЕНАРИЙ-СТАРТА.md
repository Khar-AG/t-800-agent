## Рекомендуемый путь (GitHub)

| Шаг | Действие |
|-----|----------|
| **1** | `git clone https://github.com/Khar-AG/t-800-agent.git` |
| **2** | `bash scripts/install-plugin.sh && bash scripts/verify-install.sh` |
| **3** | **Developer: Reload Window** |
| **4** | В чате: `/t800-bootstrap` |

Дальше: `/t800-doctor` → `/t800-audit` или `/t800-start`.  
Обновления — **автоматически** при новом чате (zip каждый раз не нужен).

---

## Запасной путь (zip + промпт)

| Шаг | Действие | Кто |
|-----|----------|-----|
| **1** | Папка, например `t800 agents/` | Человек |
| **2** | Положить `t-800-agent-v*.zip` | Человек |
| **3** | Open Folder → Agent → промпт ниже | Cursor |
| **4** | Reload → `/t800-bootstrap` | Человек |

### Промпт установки

```text
Установи плагин T-800 Agent из этой папки.

Сделай сам:
1. Если есть zip t-800-agent-v*.zip — разархивируй в t-800-agent.
2. bash t-800-agent/scripts/install-plugin.sh
3. bash t-800-agent/scripts/verify-install.sh — нужно verification passed.
4. Покажи version из plugin.json в ~/.cursor/plugins/local/t-800-agent/
5. Попроси Developer: Reload Window.
6. После Reload я запущу /t800-bootstrap
```

---

## Что делать после bootstrap

```text
/t800-doctor          — жив ли плагин
/t800-audit           — лишние rules/skills в Cursor
/t800-start           — создать команду / skill / агента
```

Примеры промптов: [README](../README.md#примеры-промптов).

---

## Обновление

Не кладите новый zip каждый раз. Новый чат → автопроверка GitHub → при необходимости установка → Reload.  
Срочно: `/t800-update`. Подробнее: [`ОБНОВЛЕНИЕ.md`](ОБНОВЛЕНИЕ.md).
