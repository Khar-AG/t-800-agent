# T-800 Agent — инструкция для старта

**Версия:** 1.10.0 · **Для кого:** новички

**Полная инструкция (установка + обновление + все возможности):** `T-800-ПОЛНАЯ-ИНСТРУКЦИЯ.md`


Полный сценарий в документации плагина: **`docs/СЦЕНАРИЙ-СТАРТА.md`**

---

## Что это

**T-800 Agent** — плагин Cursor для создания субагентов, skills, commands, rules и hooks через конвейер. Не нужно собирать всё руками в чате.

### Что умеет отдел (v1.10)

| Блок | Зачем |
|------|--------|
| **`/t800-audit`** | Разбор rules/skills — что жрёт контекст, что удалить |
| **`/t800-update`** | Обновить плагин со старой версии |
| Strategist + Synthesizer | Сам ищет и выбирает лучший вариант |
| Factory | skills, agents, commands, rules, hooks |

**35 субагентов.** Обновление: см. `docs/ОБНОВЛЕНИЕ.md` в архиве.


---

## Обновление со старой версии

Положите новый zip в папку → Agent → промпт:

```text
Обнови плагин T-800 Agent: разархивируй zip, bash t-800-agent/scripts/install-plugin.sh && bash t-800-agent/scripts/verify-install.sh, покажи version, попроси Reload Window.
```

Подробнее: `t-800-agent/docs/ОБНОВЛЕНИЕ.md` или команда `/t800-update` (после установки).

## Сценарий старта — 4 шага

| Шаг | Что делать |
|-----|------------|
| **1** | Создать папку, например `t800 agents` |
| **2** | Положить в неё `t-800-agent-v1.10.0.zip` |
| **3** | Open Folder в Cursor → Agent → [промпт установки](#промпт-установки) |
| **4** | Reload Window → `/t800-bootstrap` |

---

### Шаг 1 — папка

```text
t800 agents/
```

### Шаг 2 — архив

```text
t800 agents/
  └── t-800-agent-v1.10.0.zip
```

### Шаг 3 — Cursor устанавливает плагин

1. **File → Open Folder** → `t800 agents`
2. Чат **Agent**
3. Промпт ниже

### Шаг 4 — первый запуск

1. **Reload Window**
2. В чате: `/t800-bootstrap`

---

## Промпт установки

```text
В этой папке лежит архив t-800-agent-v1.10.0.zip — плагин T-800 Agent для Cursor.

Сделай всё сам:
1. Разархивируй zip в эту папку (должна появиться папка t-800-agent).
2. Установи плагин: запусти bash scripts/install-plugin.sh из папки t-800-agent, затем bash scripts/verify-install.sh — должно быть verification passed.
3. Напиши мне, что установка прошла успешно, и попроси сделать Reload Window (Developer: Reload Window).
4. После перезагрузки я напишу тебе снова — тогда запусти /t800-bootstrap.
```

---

## Промпт после Reload Window

```text
/t800-bootstrap
```

Плагин проаудирует Cursor, расскажет возможности и предложит глобальное правило (рекомендуем **«Да»**).
