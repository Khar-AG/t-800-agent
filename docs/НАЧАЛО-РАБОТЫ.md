## Что это

**T-800 Agent** — плагин Cursor: цех для создания и аудита субагентов, skills, commands, rules и hooks.  
Не нужно собирать всё руками «на глаз» в чате.

| Хочу | Команда |
|------|---------|
| Создать skill / агента / команду | `/t800-start` |
| Почистить Cursor от лишнего | `/t800-audit` |
| Проверить плагин | `/t800-plugin-audit` · `/t800-doctor` |
| Поправить по аудиту | `/t800-fix` |

**36 субагентов.** Обновление с GitHub — **автоматически** при новом чате (вручную zip каждый раз не нужен).

## Установка (один раз)

### С GitHub

```bash
git clone https://github.com/Khar-AG/t-800-agent.git
cd t-800-agent
bash scripts/install-plugin.sh
bash scripts/verify-install.sh
```

**Developer: Reload Window** → в чате:

```text
/t800-bootstrap
```

### Промпт установки (если открыли папку с zip/клоном)

```text
Установи плагин T-800 Agent из этой папки.
Разархивируй zip при необходимости, запусти install-plugin.sh и verify-install.sh,
покажи version, попроси Reload Window. После Reload я запущу /t800-bootstrap.
```

---

## Обновление

**Не нужно** каждый раз класть новый zip и жать «обновить».

При новом чате T-800 сам сверяет версию с GitHub и ставит обновление.  
После этого — **Reload Window**.

Срочно вручную: `/t800-update`

---

## Сценарий «навести порядок» (пример)

```text
1. /t800-bootstrap
2. /t800-doctor
3. /t800-audit
   → «проверь rules/skills/commands/агентов, спроси keep/narrow/remove»
4. /t800-fix
   → применить согласованные правки
5. /t800-start
   → создать нужную команду или skill
```

Примеры промптов — в [README](../README.md#примеры-промптов) и в полной инструкции.
