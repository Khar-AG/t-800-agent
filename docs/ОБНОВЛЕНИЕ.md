# Обновление T-800 с старой версии

Вы получили **новый** архив, а раньше ставили старый (1.5 / 1.6 / 1.7 / 1.8 / 1.9).

## Быстрый путь (3 шага)

1. Положите `t-800-agent-v1.13.0.zip` (или актуальный zip) в папку проекта.
2. Откройте папку в Cursor → чат **Agent** → вставьте промпт ниже.
3. **Reload Window** → проверьте `/t800-audit` или `/t800-onboard`.

## Промпт обновления (скопировать в чат)

```text
Обнови плагин T-800 Agent до актуальной версии из этой папки.

Сделай сам:
1. Если есть zip t-800-agent-v*.zip — разархивируй с обновлением папки t-800-agent.
2. bash t-800-agent/scripts/install-plugin.sh
3. bash t-800-agent/scripts/verify-install.sh — нужно verification passed.
4. Покажи version из plugin.json и сколько файлов t-800-*.md в
   ~/.cursor/plugins/local/t-800-agent/agents/
5. Попроси Reload Window.
6. После Reload я смогу использовать /t800-audit и /t800-start.
```

Или команда в чате (после установки новой версии команд):

```text
/t800-update
```

## Что появится после обновления до 1.10

- Автономный поиск (strategist / synthesizer)
- ClawHub, cookbooks GPT/Claude/Gemini/Perplexity
- **`/t800-audit`** — разбор лишних правил и контекста
- **`/t800-update`** — эта процедура
